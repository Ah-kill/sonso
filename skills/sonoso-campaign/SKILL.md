# So N So — Campaign Creator Skill

## What This Skill Does
Guides Claude to create complete marketing campaigns for So N So — covering Facebook/Instagram ads, WhatsApp outreach, landing page copy, and ad creative briefs. Built specifically for targeting medium scale business owners in India.

---

Trigger this skill when:
- "Create a Facebook ad for [industry]"
- "Write ad copy for our campaign"
- "Generate WhatsApp outreach messages"
- "Create a campaign for [restaurant / salon / gym / etc] owners"
- "Help me run ads to get clients"
- "Write a cold outreach message"

## Target Audience Profiles

### Profile 1 — The Overwhelmed Owner
- Runs a business with 5–50 employees
- Managing orders/bookings/inventory on WhatsApp + Excel
- Has heard of "software" but thinks it's too expensive or complicated
- Checks phone constantly, very mobile-dependent
- **Pain:** Chaos, missed orders, staff confusion, no visibility

### Profile 2 — The Scaling Owner
- Business growing, current manual systems breaking down
- Has tried an off-the-shelf SaaS (Zoho, etc.) but it felt too complex
- Wants something custom but was quoted ₹5L+ by an agency
- **Pain:** Wants to grow but can't systematise

### Profile 3 — The Tech-Curious Owner
- Relatively younger, comfortable with apps
- Already Googling "order management software for restaurants"
- Just needs to be convinced on trust + price
- **Pain:** Can't find something affordable that's actually built for them

## Facebook / Instagram Ad Frameworks

### Framework 1 — Pain-First (Best for cold audiences)

```
HOOK (first 3 words stop the scroll):
"Still on WhatsApp?"
"Missed order again?"
"Your staff forgot?"

BODY (identify the pain clearly):
Managing [orders/bookings/inventory] on WhatsApp is costing you customers.
Your staff misses things. You have no visibility. It's chaos at peak hours.

BRIDGE (introduce the solution):
We built a [custom order system / booking system / inventory manager]
for [restaurant/salon/retail] businesses like yours.
Ready in 7 days. You only pay infrastructure cost.

CTA:
Comment "SOFTWARE" or click the link to see a live demo.
```

### Framework 2 — Social Proof (Best for warm/retargeting audiences)

```
"[Business Name], a [city] [industry], was managing 80+ orders a day on
3 WhatsApp groups and a shared Excel file.

We built them a complete order management system in 6 days.
Their team now handles the same volume with zero missed orders.

They pay ₹800/month. That's their infrastructure cost.
We charged nothing to build it.

If you run a [industry] business in [city], we'd love to do the same for you.
DM us or click below."
```

### Framework 3 — Price Anchor (Best for owners who've been quoted high before)

```
"Got quoted ₹3 lakhs for a custom software system?

We build the same thing for you in 7 days.
You pay only infrastructure — usually ₹500–₹2,000/month.

That's it. No development fees. No contracts. No lock-ins.

We've done it for [restaurants / salons / logistics companies].
We can do it for you too.

[See a live demo → link]"
```

## Ad Creative Briefs

When generating creative briefs, always specify:

### Image Ad Brief Template
```
FORMAT: Static image (1080x1080 for feed, 1080x1920 for stories)
BACKGROUND: Dark (#0a0a0a) or brand gradient
HEADLINE TEXT: [Use one of the hooks above — max 6 words]
SUBTEXT: "7-day delivery. Infra cost only."
VISUAL ELEMENT: Screenshot of the relevant software dashboard (blurred or mockup)
LOGO: So N So logo bottom-right
CTA BUTTON: "See Demo" or "Get Yours"
TONE: Bold, clean, no clutter
```

### Video Ad Brief Template (15–30 seconds)
```
0–3s:  Hook text on screen + pain scenario (e.g., WhatsApp chaos)
3–10s: Screen recording of the software in action (dashboard, placing order)
10–20s: "Built in 7 days. You pay only infrastructure."
20–30s: CTA + So N So branding

VOICEOVER TONE: Calm, confident, not salesy
TEXT ON SCREEN: Key stats (7 days, ₹800/month, 0 development fee)
```

## Campaign Structure (Facebook Ads Manager)

```
CAMPAIGN OBJECTIVE: Leads or Messages
BUDGET: ₹200–₹500/day to start

AD SET 1 — Cold (Top of Funnel)
  Audience: Business Owners, 28–55, Tier 1/2 cities
  Interests: [Industry-specific] + Entrepreneurship + Business Management
  Ad: Pain-First framework

AD SET 2 — Interest (Mid Funnel)
  Audience: People who engaged with your page / watched 50%+ of a video
  Ad: Social Proof framework

AD SET 3 — Retargeting (Bottom of Funnel)
  Audience: Website visitors (last 30 days)
  Ad: Price Anchor framework

CONVERSION: WhatsApp message OR lead form (Name + WhatsApp number + Business type)
```

## Output Format

1. `campaign-brief.md` — target audience, objective, budget
2. `ad-copy.md` — 3 ad variations using the frameworks above
3. `creative-brief.md` — image/video specs for each ad
4. `whatsapp-outreach.md` — cold + follow-up messages
5. `campaign-checklist.md` — pre-launch QA checklist

---

## Tone Rules

- **Never use corporate jargon.** Write like a smart friend explaining something.
- **Be specific.** "₹800/month" beats "affordable pricing."
- **Lead with pain, not features.** The owner doesn't care about "admin dashboards" — they care about not missing orders.
- **Short sentences.** Business owners skim. Make every line earn its place.
