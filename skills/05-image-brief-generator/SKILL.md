# So N So — Image Brief Generator Skill

## What This Skill Does
Generates precise image briefs and AI image prompts for So N So marketing assets — ads, social posts, website banners, thumbnails, and logo concepts. Every brief is aligned with the So N So visual identity.

## Trigger Phrases
- "Generate an image brief for [ad/post/banner]"
- "Create a prompt for [AI image tool] for [asset type]"
- "Write a logo brief"
- "Describe the image for [campaign name]"

---

## So N So Visual Identity (Always Apply)

| Element | Specification |
|---|---|
| Background | Dark: #0a0a0a to #161616 |
| Accent colour | Emerald green: #10b981 |
| Text colour | White (#ffffff) or off-white (#f0f0f0) |
| Font | Inter — bold/black weight for headlines |
| Style | Clean, minimal, premium tech |
| Mood | Confident, fast, modern — not playful, not corporate |
| Never use | Blues, oranges, clipart, stock suit photos, gradients with multiple colours |

```
TEXT ON IMAGE:
- Headline: "[text]"

COLOUR TREATMENT:
- Background: [dark / card / gradient — specify hex]
- Accent: Emerald green #10b981

[e.g., "Relief — the business owner sees clarity for the first time"]

DO NOT INCLUDE:
[Specific elements to avoid for this asset]
```

## Logo Brief (Full)

**Company:** So N So (stylised: SoNSo)
**Industry:** Custom software / tech services
**Audience:** SME business owners in India
**Feeling the logo should give:** Fast, trustworthy, modern, approachable but professional

**Primary logo concept:**
Bold wordmark "SoNSo" using Inter Black or similar weight. The letter "N" in the centre should be:
- Option A: Filled with emerald green (#10b981)
- Option B: Enclosed in a green rectangle/box
- Option C: Replaced with a lightning bolt or arrow icon

**Secondary icon concept:**
A single geometric node — a dot with two connecting lines flowing forward (like a process being connected) — in emerald green. Used as standalone app icon or favicon.

**Deliverables needed:**
- Full wordmark (horizontal) — light version (on dark BG) + dark version (on light BG)
- Icon only (for favicon, app icon, social profile picture)
- Combined icon + wordmark
- Single colour version (all white, all black)

**Files needed:** SVG, PNG (transparent), 512x512 icon
