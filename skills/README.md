# So N So — Skills Directory

All skills are organised by number and function. Each folder contains a SKILL.md file.
Always read the relevant SKILL.md before starting any task.

Also read SONOSO_CONTEXT.md in the parent folder before using any skill —
it contains the full brand, audience, pricing, and tone guidelines.
